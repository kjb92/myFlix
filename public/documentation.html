Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user

Parameter Type Description
title Text 
e.g. "Frozen", "The Lion King"
Request body data format JSON 
None
Response body data format JSON 
                [
                  {
                    "title": "Text",
                    "description": "Text",
                    "director": {
                        "firstName": "Text",
                        "lastName": "Text",

GET /movies/genre/{genreName}

Return data about a genre (description) by name/title

Parameter Type Description
genreName Text 
e.g. "Animation", "Drama"
Request body data format JSON 
None
Response body data format JSON 
                {
                  "name": "Animation",
                  "description": "..."
                }

GET /movies/director/{directorName}

Return data about a director (bio, birth year, death year) by name

Parameter Type Description
directorName Text 
e.g. "Roger Allers"
Request body data format JSON 
None
Response body data format JSON 
                {
                  "firstName": "Roger",
                  "lastName": "Allers",
                  "bio": "Roger Allers is an American film director, screenwriter, storyboard artist, animator and voice actor. He is best known for co-directing the Disney animated feature The Lion King (1994).",
                  "dateOfBirth": 1949
                }

POST /users

Allow new users to register

Parameter Type Description
Request body data format JSON 
                {
                  "username": "marymason",
                  "firstName": "Mary",
                  "lastName": "Mason",
                  "favoriteMovies":[]
                }
Response body data format JSON 
                {
                  "username": "marymason",
                  "firstName": "Mary",
                  "lastName": "Mason",
                  "favoriteMovies": [],
                  "id": "8daa9319"
              }

POST /users/{id}/{movieTitle}

Allow users to add a movie to their list of favorites

Parameter Type Description
id Text 
e.g. "8daa9319"
movieTitle Text 
e.g. "The Lion King"
Request body data format JSON 
None
Response body data format Text 
The Lion King has been added to user 1's favorite movies

PUT /users/{id}

Allow users to update their user info

Parameter Type Description
id Text 
e.g. "1"
Request body data format JSON 
                {
                  "username": "kevin-updated"
                }
Response body data format JSON 
                {
                  "id": 1,
                  "username": "kevin-updated",
                  "firstName": "Kevin",
                  "lastName": "Blumenstock",
                  "favoriteMovies": [
                      "The Lion King"
                  ]
                }

DELETE /users/{id}/{movieTitle}

Allow users to remove a movie from their list of favorites

Parameter Type Description
id Text 
e.g. "2"
movieTitle Text 
e.g. "The Lion King"
Request body data format JSON 
None
Response body data format Text 
The Lion King has been removed from user 2's favorite movies

DELETE /users/{id}

Allow existing users to deregister

Parameter Type Description
id Text 
e.g. "3"
Request body data format JSON 
None
Response body data format Text 
The user with username "deleteme" and user-id "3" has been deleted
