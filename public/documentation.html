POST /users

Allow new users to register

Parameter Type Description
Request body data format JSON 
{
  "username": "marymason2",
  "email": "mary@test.com",
  "password": "testpassword12345",
  "birthday": "2000-01-01"
}
Response body data format JSON 
{
  "username": "marymason2",
  "password": "testpassword12345",
  "email": "mary@test.com",
  "birthday": "2000-01-01T00:00:00.000Z",
  "favoriteMovies": [],
  "_id": "644926b635e98a9cb61d9b96",
  "__v": 0
}

POST /users/{username}/movies/{movieID}

Allow users to add movies to their list of favorite movies

Parameter Type Description
username Text 
e.g. "marymason"
movieID Text 
e.g. "64462f40a5f5577c2975ae7b"
Request body data format JSON 
None
Response body data format Text 
{
  "_id": "64491747cbbcaca8d7c4e944",
  "username": "marymason",
  "password": "testpassword12345",
  "email": "mary@test.com",
  "favoriteMovies": [
      "64462f40a5f5577c2975ae7b"
  ],
  "__v": 0
}

      "64462f40a5f5577c2975ae7b"

Parameter Type Description
